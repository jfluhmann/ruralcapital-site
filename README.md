# Rural Capital, LLC - Landing Page

Static landing page for [ruralcapital.co](https://ruralcapital.co).

## Stack

Plain HTML, CSS, and vanilla JavaScript. No build tools, no framework.

## Development

Open `index.html` in a browser, or use any local server:

```bash
python3 -m http.server 8000
```

## Deployment

Deployed via Netlify from this GitHub repo. Push to `main` triggers automatic deploy.

- Publish directory: `.` (root)
- No build command required
