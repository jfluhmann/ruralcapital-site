# Rural Capital Landing Page

## Project Type
Static HTML/CSS/JS landing page. No build tools, no framework.

## File Structure
- `index.html` — Single-page site with all content
- `css/style.css` — All styles (CSS custom properties, no preprocessor)
- `js/main.js` — Vanilla JS (scroll reveal, nav, mobile menu)
- `assets/` — SVGs, favicon, static assets

## Design Conventions
- Color palette defined as CSS custom properties on `:root`
- Fonts: Playfair Display (headings), Source Sans 3 (body) via Google Fonts
- Mobile-first responsive design
- Parchment background (#f5f0ea), not pure white

## Deployment
- Netlify auto-deploys from `main` branch
- Domain: ruralcapital.co
- No build step required
